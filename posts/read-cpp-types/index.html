еј•иЁҖ#

C++ зҡ„зұ»еһӢеҸҜиҜ»жҖ§еҫҲе·®пјҢе№¶дё”еӨ§еӨҡж•°е…Ҙй—Ёжқҗж–ҷдёӯе№¶жІЎжңүиҜҰз»Ҷд»Ӣз»ҚеҰӮдҪ•йҳ…иҜ»е®ғд»¬пјҢжңҖеӨҡеҸӘжҳҜи®ІеҲ° top-level const е’Ң low-level const зҡ„еҢәеҲ«гҖӮжңүдёҚе°‘жңӢеҸӢй—®иҝҮжҲ‘иҝҷж–№йқўзҡ„й—®йўҳпјҢи®Іеҫ—еӨҡдәҶпјҢе№Іи„Ҷж•ҙзҗҶиө·жқҘеҶҷзҜҮж–Үз« гҖӮ

еёёи§ҒиҜҜи§Ј#

еңЁиҜҰз»Ҷи®Ізұ»еһӢзҡ„йҳ…иҜ»д№ӢеүҚпјҢйңҖиҰҒзә жӯЈдёҖдёӘеёёи§Ғзҡ„иҜҜи§ЈгҖӮ

й—®пјҡint a[5]йҮҢзҡ„aжҳҜд»Җд№Ҳзұ»еһӢзҡ„пјҹ

зӯ”пјҡint[5]зұ»еһӢпјҢеңЁйҖӮеҪ“зҡ„ж—¶еҖҷдјҡгҖҢйҖҖеҢ–гҖҚпјҲdecayпјүжҲҗдёәint*зұ»еһӢгҖӮ

й—®пјҡint a[5][6]йҮҢзҡ„aжҳҜд»Җд№Ҳзұ»еһӢзҡ„пјҹ

зӯ”пјҡint[5][6]зұ»еһӢпјҢеңЁйҖӮеҪ“зҡ„ж—¶еҖҷдјҡгҖҢйҖҖеҢ–гҖҚжҲҗдёәint(*)[6]зұ»еһӢпјҢеҚіжҢҮеҗ‘int[6]зҡ„жҢҮй’ҲгҖӮ

з”ұдәҺгҖҢйҖҖеҢ–гҖҚиҝҷз§ҚйҡҗејҸиҪ¬жҚўзҡ„еӯҳеңЁпјҢеҫҲеӨҡеҲқеӯҰ C++ зҡ„дәәдјҡжҠҠж•°з»„зұ»еһӢзӯүеҗҢдәҺжҢҮй’Ҳзұ»еһӢгҖӮ

зұ»дјјзҡ„пјҢеҮҪж•°зұ»еһӢд№ҹдјҡгҖҢйҖҖеҢ–гҖҚеҲ°жҢҮй’Ҳзұ»еһӢпјҢеҰӮint(int, int)дјҡгҖҢйҖҖеҢ–гҖҚжҲҗдёәint(*)(int, int)гҖӮ

и§Јж–№зЁӢ#

еҪ“дҪ жҹҘйҳ…еҰӮдҪ•йҳ…иҜ»дёҖдәӣеӨҚжқӮзҡ„зұ»еһӢж—¶пјҢдҪ еҸҜиғҪдјҡзңӢеҲ°зҪ‘з»ңдёҠдёҖдәӣдәәиҜҙ C++ зҡ„зұ»еһӢе°ұжҳҜи§Јж–№зЁӢпјҢи®©жҲ‘жқҘиҜҰз»Ҷи§ЈйҮҠдёҖдёӢиҝҷеҸҘиҜқгҖӮ

жҠӣеҺ» CVR (const, volatile, reference) зӯүдёҚи°ҲпјҢC++ жңҖеҹәжң¬зҡ„еЈ°жҳҺеҲҶдёәдёӨдёӘйғЁеҲҶпјҡеҶҷеңЁжңҖе·Ұиҫ№зҡ„зұ»еһӢеҗҚжҳҜгҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚпјҲtype specifierпјүпјҢеү©дёӢзҡ„йғЁеҲҶжҳҜгҖҢеЈ°жҳҺз¬ҰгҖҚпјҲdeclaratorпјүгҖӮиҝҷйҮҢжІҝз”ЁдәҶ C иҜӯиЁҖдёӯзҡ„иҜҙжі•пјҢеӣ дёәиҝҷйғЁеҲҶеҶ…е®№жҳҜд»Һ C з»§жүҝиҝҮжқҘзҡ„пјҢе®ғд»¬жҳҜ C++ зұ»еһӢйҮҢжңҖжҒ¶еҝғзҡ„ең°ж–№гҖӮдҪҶиҝҷдёӨдёӘеҗҚеӯ—е®һеңЁжІЎе•ҘиҜҶеҲ«еәҰпјҢжҲ‘е–ңж¬ўдёҚдёҘи°Ёең°з§°е‘јдёәиҝ”еӣһеҖјзұ»еһӢе’Ңи°ғз”ЁиЎЁиҫҫејҸпјҢиҝҷзӮ№еҗҺйқўдјҡи§ЈйҮҠгҖӮе…ҲжқҘзңӢеҮ дёӘдҫӢеӯҗеҗ§пјҡ

еЈ°жҳҺзұ»еһӢиҜҙжҳҺз¬ҰеЈ°жҳҺз¬Ұ
int ainta
int* aint*a
int a[5]inta[5]
int* a[5]int*a[5]
int (*a)[5]int(*a)[5]

дҪңеҮәиҝҷз§ҚеҲҶеҲ«еҗҺпјҢе°ұеҸҜд»ҘзҗҶи§ЈдёҠиЎЁдёӯaзҡ„зұ»еһӢжҳҜжҖҺж ·еҶіе®ҡзҡ„дәҶпјҡд»ҘгҖҢеЈ°жҳҺз¬ҰгҖҚзҡ„еҪўејҸи°ғз”ЁaеҗҺпјҢеҫ—еҲ°зҡ„иҝ”еӣһеҖјзұ»еһӢдёәгҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚгҖӮжүҖи°“зҡ„и§Јж–№зЁӢе°ұжҳҜиҝҷж ·дёҖдёӘиҝҮзЁӢпјҡ

  1. з”ұint *a[5]еҫ—еҲ°*a[5]зҡ„зұ»еһӢжҳҜint
  2. еҚіеҜ№a[5]и§Јеј•з”Ёеҫ—еҲ°intпјҢжҺЁеҮәa[5]зҡ„зұ»еһӢжҳҜint*
  3. еҚіеҜ№aеҸ–дёӢж Үеҫ—еҲ°int*пјҢжҺЁеҮәaзҡ„зұ»еһӢжҳҜ 5 дёӘint*зҡ„ж•°з»„

еҶҚжҜ”еҰӮпјҡ

  1. з”ұint (*a)[5]еҫ—еҲ°(*a)[5]зҡ„зұ»еһӢжҳҜint
  2. еҚіеҜ№(*a)еҸ–дёӢж Үеҫ—еҲ°intпјҢжҺЁеҮә(*a)зҡ„зұ»еһӢжҳҜint[5]
  3. еҚіеҜ№aи§Јеј•з”Ёеҫ—еҲ°int[5]пјҢжҺЁеҮәaзҡ„зұ»еһӢжҳҜжҢҮеҗ‘int[5]зҡ„жҢҮй’Ҳ

еҸҜд»ҘзңӢеҲ°иҝҷйҮҢиҝҗз®—з¬Ұдјҳе…Ҳзә§е’Ңи°ғз”Ёж—¶е®Ңе…ЁдёҖиҮҙпјҢеҪўејҸдёҠд№ҹеҹәжң¬дёҖиҮҙгҖӮ

еҶҚжқҘдёӘеҮҪж•°жҢҮй’Ҳзҡ„дҫӢеӯҗпјҡ

  1. з”ұint (*a)(int, int)еҫ—еҲ°(*a)(int, int)зҡ„зұ»еһӢжҳҜint
  2. еҚіеҜ№(*a)д»Ҙ(int, int)зҡ„еҪўејҸи°ғз”Ёеҫ—еҲ°intпјҢжҺЁеҮә(*a)зҡ„зұ»еһӢжҳҜжңүдёӨдёӘintеҸӮж•°е№¶дё”иҝ”еӣһintзҡ„еҮҪж•°пјҢеҚіint(int, int)
  3. еҚіеҜ№aи§Јеј•з”Ёеҫ—еҲ°int(int, int)пјҢжҺЁеҮәaзҡ„зұ»еһӢжҳҜжҢҮеҗ‘int(int, int)зҡ„жҢҮй’Ҳ

жӣҙеӨҚжқӮдёҖзӮ№пјҡ

  1. з”ұint (*(Foo::*a[5])(int, int))(int)еҫ—еҲ°(*(Foo::*a[5])(int, int))(int)зҡ„зұ»еһӢжҳҜint
  2. жҺЁеҮә(*(Foo::*a[5])(int, int))зҡ„зұ»еһӢжҳҜint(int)
  3. жҺЁеҮә(Foo::*a[5])(int, int)зҡ„зұ»еһӢжҳҜжҢҮеҗ‘int(int)зҡ„жҢҮй’ҲпјҢеҚіint(*)(int)
  4. жҺЁеҮә(Foo::*a[5])зҡ„зұ»еһӢжҳҜжңүдёӨдёӘintеҸӮж•°е№¶дё”иҝ”еӣһint(*)(int)зҡ„еҮҪж•°
  5. жҺЁеҮәa[5]зҡ„зұ»еһӢжҳҜжҢҮеҗ‘иҝҷз§ҚеҮҪж•°зҡ„жҲҗе‘ҳжҢҮй’Ҳ
  6. жҺЁеҮәaзҡ„зұ»еһӢжҳҜ 5 дёӘиҝҷз§ҚжҲҗе‘ҳжҢҮй’Ҳзҡ„ж•°з»„

з®ҖиҖҢиЁҖд№ӢпјҢдҪ еҸҜд»ҘжҠҠгҖҢеЈ°жҳҺз¬ҰгҖҚйғЁеҲҶзңӢжҲҗдёҖдёӘи°ғз”ЁиЎЁиҫҫејҸпјҢиҝҷдёӘи°ғз”ЁиЎЁиҫҫејҸзҡ„иҝ”еӣһеҖјзұ»еһӢдёәгҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚпјҢиҝҷж ·е°ұеҸҜд»ҘдёҖжӯҘжӯҘеҖ’жҺЁеҮәзұ»еһӢжқҘгҖӮ

еҪ“з„¶дәҶпјҢиҝҷд№ҲеӨҚжқӮзҡ„зұ»еһӢжҳҜдёҚжҺЁиҚҗиҝҷд№ҲеҶҷзҡ„пјҢиҝҷйҮҢеҸӘжҳҜдёәдәҶжј”зӨәдёҖдёӢи§Јж–№зЁӢзҡ„иҝҮзЁӢгҖӮдёҠйқўйӮЈдёӘaжӯЈеёёдёҖзӮ№зҡ„еЈ°жҳҺеӨ§жҰӮдјҡй•ҝиҝҷж ·пјҡ

5

struct Foo {
    auto bar(int, int) -> int(*)(int);
};

decltype(&Foo::bar) a[5];

foo_bar_t a[5];

иҝҷз§ҚеҢ…иЈ…еңЁ C иҜӯиЁҖдёӯиҝҳжҳҜжҢәеёёи§Ғзҡ„гҖӮ

дёҚиҝҮеҰӮжһңдҪ иҰҒзңӢзҡ„зұ»еһӢжҳҜ demangle еҮәжқҘзҡ„пјҲжҲ–иҖ…е…¶д»–жҹҘзңӢзұ»еһӢеҗҚзҡ„ж–№ејҸпјүпјҢйӮЈе°ұжҳҜй•ҝжҲҗйӮЈз§Қйӯ”й¬јж ·еӯҗдәҶгҖӮ

еӨҡеҸҳйҮҸеЈ°жҳҺ#

жңүдәҶеүҚйқўзҡ„зҹҘиҜҶпјҢе°ұиғҪеӨҹзҗҶи§Ј C++ еӨҡеҸҳйҮҸеЈ°жҳҺзҡ„и§„еҲҷдәҶгҖӮеӨҡеҸҳйҮҸеЈ°жҳҺдёӯпјҢйҖ—еҸ·еҲҶеүІзҡ„жҳҜгҖҢеЈ°жҳҺз¬ҰгҖҚпјҲеҚіи°ғз”ЁиЎЁиҫҫејҸйғЁеҲҶпјүпјҢе®ғд»¬е…ұдә«гҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚпјҲеҚіиҝ”еӣһеҖјйғЁеҲҶпјүпјҢжҜ”еҰӮпјҡ

1

int *a, b;

е…¶дёӯaзҡ„зұ»еһӢдёәint*иҖҢbзҡ„зұ»еһӢеҲҷдёәintпјҢеӣ дёә*жҳҜгҖҢеЈ°жҳҺз¬ҰгҖҚзҡ„дёҖйғЁеҲҶгҖӮеҰӮжһңдҪ еёҢжңӣе®ҡд№үдёӨдёӘжҢҮй’ҲпјҢйӮЈд№Ҳеә”иҜҘеҶҷжҲҗint *a, *bгҖӮ

жӣҙеӨҚжқӮзҡ„дҫӢеӯҗд№ҹйҒөеҫӘеҗҢж ·зҡ„и§„еҲҷгҖӮеӣ дёәе®һеңЁжғідёҚеҲ°жӣҙеӨҚжқӮзҡ„жғ…еҶөиҝҷж ·еҶҷжңүд»Җд№Ҳж„Ҹд№үпјҢе°ұдёҚеҶҚдёҫдҫӢеӯҗдәҶгҖӮ

constдёҺvolatile#

constдёҺvolatileеңЁзұ»еһӢдёӯзҡ„дҪҚзҪ®жҳҜдёҖж ·зҡ„пјҢе№¶дё”д»–д»¬дёӨдёӘеҸҜд»ҘдёҖиө·дҪҝз”ЁпјҢеӣ жӯӨиҝҷйҮҢд»…д»…з”ЁconstдҪңдёәдҫӢеӯҗгҖӮдёӢйқўеҮәзҺ°зҡ„жүҖжңүconstйғҪеҸҜд»ҘеҗҲжі•ең°жӣҝжҚўжҲҗvolatileгҖҒconst volatileе’Ңvolatile constпјҢеҗҺдёӨиҖ…иҜӯд№үдёҠзӯүд»·гҖӮ

constеңЁдёҖиҲ¬жғ…еҶөдёӢйғҪеҫҲжҳҺзЎ®пјҢеҰӮconst int aе’Ңint (*a)(const std::string&)гҖӮconstзҡ„зҗҶи§ЈйҡңзўҚдё»иҰҒеҮәеңЁжҢҮй’Ҳзҡ„жғ…еҶөдёӯгҖӮеҜ№дәҺжҢҮй’ҲжқҘиҜҙпјҢconstйңҖиҰҒиғҪиЎЁиҫҫдёӨз§Қеҗ«д№үпјҡ

  1. жҢҮй’Ҳжң¬иә«жҳҜconstзҡ„пјҢеҚіиҝҷдёӘжҢҮй’ҲжІЎжі•еҶҚжҢҮеҗ‘еҲ«зҡ„еҖјпјҢеҰӮchar *const aгҖӮ
  2. жҢҮй’ҲжүҖжҢҮеҗ‘зҡ„еҜ№иұЎжҳҜconstзҡ„пјҢеҚідҪ жІЎжі•йҖҡиҝҮиҝҷдёӘжҢҮй’Ҳдҝ®ж”№е®ғжҢҮеҗ‘зҡ„еҜ№иұЎпјҢеҰӮconst char* aгҖӮ

еҪ“жҢҮй’ҲеөҢеҘ—ж—¶пјҢжҲ‘д»¬иҝҳиҰҒиҖғиҷ‘жҢҮй’ҲжҢҮеҗ‘зҡ„жҢҮй’Ҳзҡ„constеұһжҖ§пјҢдәҺжҳҜзұ»еһӢе°ұеҸҳеҫ—еӨҚжқӮиө·жқҘгҖӮжңүдәҶеүҚйқўзҡ„зҹҘиҜҶпјҢжҲ‘д»¬еҸҜд»ҘжҠҠconstеҲҶдёәдҝ®йҘ°гҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚзҡ„constе’Ңдҝ®йҘ°гҖҢеЈ°жҳҺз¬ҰгҖҚзҡ„constгҖӮзңӢдёӢйқўиҝҷдёӘзұ»еһӢпјҡ

1

const int * const * * const a;

д»Һе·ҰеҫҖеҸізҡ„з¬¬дёҖдёӘconstдёәдҝ®йҘ°гҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚзҡ„constпјҢе…¶дҪҷзҡ„дёәдҝ®йҘ°гҖҢеЈ°жҳҺз¬ҰгҖҚзҡ„constгҖӮ

дҝ®йҘ°гҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚзҡ„constеҸӘиҰҒеҶҷеңЁгҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚж—Ғиҫ№пјҢдёҚи®әconst TиҝҳжҳҜT constйғҪжҳҜеҗҲжі•дё”зӯүд»·зҡ„гҖӮд№ҹе°ұжҳҜдёҠйқўиҝҷдёӘдҫӢеӯҗиҝҳеҸҜд»ҘеҶҷжҲҗпјҡ

1

int const * const * * const a;

е®ғд»¬йғҪиҜҙжҳҺпјҡ

  1. * const * * const aпјҲеҪ“з„¶еҺ»жҺүconstеҗҺе®ғжүҚжҳҜдёҖдёӘеҗҲжі•зҡ„иЎЁиҫҫејҸпјҢдёӢеҗҢпјүзҡ„зұ»еһӢдёәconst intпјҲдёҚиғҪдҝ®ж”№***aпјү

дҝ®йҘ°гҖҢеЈ°жҳҺз¬ҰгҖҚзҡ„constеҸӘиҰҒзңӢи§Јж–№зЁӢж—¶и§ЈеҲ°е“ӘдёӘиЎЁиҫҫејҸпјҢе°ұиҜҙжҳҺдҝ®йҘ°зҡ„жҳҜе“ӘдёӘиЎЁиҫҫејҸзҡ„зұ»еһӢгҖӮжҺҘзқҖдёҠйқўиҝҷдёӘдҫӢеӯҗпјҡ

  1. жҺЁеҮә* * const aжҳҜдёҖдёӘжҢҮеҗ‘const intзҡ„ const жҢҮй’ҲпјҲдёҚиғҪдҝ®ж”№**aпјү
  2. жҺЁеҮә* const aжҳҜдёҖдёӘжҢҮеҗ‘еүҚйқўйӮЈдёӘжҢҮй’Ҳзҡ„йқһ const жҢҮй’ҲпјҲеҸҜд»Ҙдҝ®ж”№*aпјү
  3. жҺЁеҮәaжҳҜдёҖдёӘжҢҮеҗ‘еүҚйқўйӮЈдёӘжҢҮй’Ҳзҡ„ const жҢҮй’ҲпјҲдёҚиғҪдҝ®ж”№aпјү

и‘—еҗҚзҡ„ C++ е…Ҙй—Ёд№ҰзұҚгҖҠC++ PrimerгҖӢжҠҠconstеҲҶдёәдәҶгҖҢйЎ¶еұӮ const гҖҚпјҲtop-level constпјүе’ҢгҖҢеә•еұӮ const гҖҚпјҲlow-level constпјүпјҲжӯӨдёәеҺҹзүҲзҝ»иҜ‘пјҢжҳҫз„¶е№¶дёҚеҮҶзЎ®пјүгҖӮйЎ¶еұӮжҢҮзҡ„жҳҜдҝ®йҘ°жҢҮй’ҲиҮӘиә«зҡ„пјҢд№ҹе°ұжҳҜзұ»еһӢдёӯжңҖеҶ…еұӮзҡ„пјӣеә•еұӮжҢҮзҡ„жҳҜе…¶дҪҷзҡ„гҖӮиҝҷдёӨдёӘжңҜиҜӯеңЁеҸӘи®Ёи®әдёҖеұӮжҢҮй’Ҳзҡ„ж—¶еҖҷжүҚжҜ”иҫғеҘҪз”ЁпјҢеҪ“з„¶иҝҷд№ҹжҳҜз»қеӨ§еӨҡж•°жғ…еҶөгҖӮ

&дёҺ&&#

C++ зҡ„еј•з”ЁжҳҜдёҖз§Қйқһеёёзү№ж®Ҡзҡ„зұ»еһӢгҖӮз”ұдәҺ C++ и§„е®ҡеј•з”ЁдёҚдёҖе®ҡеҚ з”ЁеҶ…еӯҳпјҢжүҖд»ҘдҪ ж—ўжІЎжі•еҶҷеҮәеј•з”Ёзұ»еһӢзҡ„ж•°з»„пјҲint &a[5]пјүпјҢд№ҹжІЎжі•еҶҷеҮәжҢҮеҗ‘еј•з”Ёзұ»еһӢзҡ„жҢҮй’ҲпјҲint&* aпјүпјҢйҷӨжӯӨд№ӢеӨ–иҝҳжңүеҫҲеӨҡең°ж–№ж— жі•дҪҝз”Ёеј•з”ЁгҖӮжҲ‘и®ӨдёәиҝҷжҳҜдёҖдёӘйқһеёёеӨұиҙҘзҡ„и®ҫи®ЎпјҢеҮӯз©әеўһеҠ дәҶеҫҲеӨҡйҡңзўҚгҖӮе®ғжҳҺжҳҺеҸӘжҳҜдёҖдёӘгҖҢдёҚеҸҜз©әжҢҮй’ҲгҖҚпјҲnon-nullable pointerпјүиҖҢе·ІгҖӮеҸіеҖјеј•з”Ёд№ҹе№¶жІЎжңүд»Җд№Ҳзү№еҲ«пјҢеҸӘжҳҜеёҰжңүжүҖжңүжқғиҪ¬з§»иҜӯд№үзҡ„еј•з”ЁиҖҢе·ІгҖӮ

жҖ»д№ӢпјҢйҷӨдәҶеҮҪж•°зұ»еһӢпјҲд»ҘеҸҠеөҢеҘ—дәҶеҮҪж•°зұ»еһӢзҡ„зұ»еһӢпјүйҮҢ&е’Ң&&дјҡеҮәзҺ°еңЁеҸӮж•°зұ»еһӢе’Ңиҝ”еӣһеҖјзұ»еһӢдёӯпјҢе…¶д»–жғ…еҶөдёӢ&е’Ң&&еҸӘдјҡеҮәзҺ°еңЁзұ»еһӢзҡ„жңҖйЎ¶еұӮгҖӮиҝҷйҮҢзҡ„йЎ¶еұӮзҡ„еҗ«д№үдёҺеүҚж–Үзҡ„гҖҢйЎ¶еұӮ const гҖҚзӣёеҗҢгҖӮ

&е’Ң&&жІЎжі•зӣҙжҺҘеөҢеҘ—дҪҝз”ЁпјҢжҜ”еҰӮint& && aжҳҜйқһжі•зҡ„гҖӮдҪҶжҳҜй—ҙжҺҘеөҢеҘ—жҳҜеҗҲжі•зҡ„пјҢжҜ”еҰӮпјҡ

2
3

using ref1 = int&&;
using ref2 = ref1&&;
using ref3 = ref2&;

еңЁиҝҷз§Қжғ…еҶөдёӢдјҡеҸ‘з”ҹгҖҢеј•з”ЁжҠҳеҸ гҖҚпјҲreference collapsingпјүгҖӮи§„еҲҷд№ҹеҫҲз®ҖеҚ•пјҢеҰӮжһңе…ЁжҳҜ&&пјҢйӮЈд№ҲдјҡжҠҳеҸ жҲҗдёҖдёӘ&&гҖӮеҸӘиҰҒжңүдёҖдёӘ&пјҢе°ұдјҡжҠҳеҸ жҲҗ&гҖӮжҜ”еҰӮдёҠйқўзҡ„ref2е®һйҷ…дёҠжҳҜint&&пјҢиҖҢref3еҲҷжҳҜint&гҖӮ

&&иҝҳжңүдёҖз§Қз”Ёжі•жҳҜеңЁжЁЎжқҝдёӯгҖӮеҰӮжһңTжҳҜдёҖдёӘжЁЎжқҝеҸӮж•°пјҢйӮЈд№ҲT&&пјҲдёҚиғҪеёҰжңүconstе’Ңvolatileпјүе°ұжҳҜдёҖдёӘгҖҢдёҮиғҪеј•з”ЁгҖҚпјҲuniversal referenceпјүгҖӮе…·дҪ“и§„еҲҷдёәпјҡ

  1. еҰӮжһңдј е…ҘT&&зҡ„зұ»еһӢдёәFoo&пјҢйӮЈд№ҲTе°ұиў«жҺЁеҜјдёәFoo&пјҢиҖҢT&&з»ҸиҝҮжҠҳеҸ еҸҳжҲҗFoo&гҖӮ
  2. еҰӮжһңдј е…ҘT&&зҡ„зұ»еһӢдёәFoo&&пјҢйӮЈд№ҲTе°ұиў«жҺЁеҜјдёәFooпјҢиҖҢT&&еҲҷжҳҜFoo&&гҖӮ

иҜ·жіЁж„ҸпјҢTиў«жҺЁеҜјдёәд»Җд№Ҳзұ»еһӢиҝҷдёҖзӮ№еҫҲйҮҚиҰҒпјҢеӣ дёәдҪ еҸҜиғҪеңЁжЁЎжқҝеҶ…йғЁеҶҚж¬ЎдҪҝз”ЁTгҖӮ

жҲҗе‘ҳеҮҪж•°#

еңЁ C++ зҡ„жҲҗе‘ҳеҮҪж•°дёӯпјҢжҲ‘д»¬жңүж—¶дјҡи§ҒеҲ°иҝҷж ·зҡ„дҫӢеӯҗпјҡ

еңЁе…¶д»–дёҖдәӣиҜӯиЁҖпјҲжҜ”еҰӮ Rust е’Ң Python пјүдёӯдј е…Ҙзҡ„е®һдҫӢжҳҜжҳҫејҸең°дҪңдёәз¬¬дёҖдёӘеҸӮж•°еҶҷеҮәзҡ„пјҢиҖҢ C++ жҳҜйҡҗејҸзҡ„пјҢдәҺжҳҜеҸӘиғҪеңЁе…¶д»–ең°ж–№еҠ дёҠзұ»еһӢдҝ®йҘ°гҖӮдёҚиҝҮеҚідҪҝдёҚжҳҜйҡҗејҸзҡ„пјҢдҪ д№ҹжІЎжі•еңЁ C++ дёӯзӣҙжҺҘеҶҷеҮәthisзҡ„зұ»еһӢпјҢдҪҶиҝҷд»Қз„¶жҳҜ C++ зҡ„й”…вҖ”вҖ”еӣ дёәthisжҳҜдёҖдёӘжҢҮй’ҲпјҢиҖҢ C++ дёӯжҢҮй’ҲжІЎжі•жҢҮеҗ‘еј•з”Ёзұ»еһӢгҖӮдёӢйқўдёӨдёӘдҫӢеӯҗз”ЁжіЁйҮҠдёҚдёҘи°Ёең°жҸӯзӨәдәҶеҗҺзҪ®зҡ„зұ»еһӢдҝ®йҘ°жҳҜжҖҺд№ҲиҝҗдҪңзҡ„пјҡ

д»ҺиҝҷдёӘдҫӢеӯҗдҪ д№ҹеҸҜд»ҘеҶҚдёҖж¬ЎзңӢеҲ° C++ зҡ„еј•з”Ёи®ҫи®Ўеҫ—еӨҡд№ҲеӨұиҙҘгҖӮ

жӣҙеӨҡ#

C++ зҡ„еЈ°жҳҺиҜӯеҸҘжһ„жҲҗйғЁеҲҶзӣёеҪ“еӨҚжқӮпјҢе№¶йқһеҸӘжңүгҖҢзұ»еһӢиҜҙжҳҺз¬ҰгҖҚе’ҢгҖҢеЈ°жҳҺз¬ҰгҖҚпјҢдҪҶе…¶дҪҷйғЁеҲҶзӣёжҜ”дёҠж–ҮжҸҗеҲ°зҡ„йӮЈдәӣе·Із»ҸзӣёеҪ“дәәзұ»еҸӢеҘҪдәҶпјҢжІЎжңүеӨҡе°‘еӯҰд№ йҡңзўҚпјҢиҖҢдё”е…¶дёӯзҡ„еҫҲеӨҡдёҺзұ»еһӢж— е…ігҖӮжңүе…ҙи¶ЈдәҶи§Јзҡ„еҸҜд»ҘеҸӮйҳ… cppreference гҖӮ
